> HandsUp UI Toolkit

## Install
```shell
yarn add handsup-ui
# npm install element-ui -S
```

## Quick Start
```shell
import Vue from 'vue'
import handsupui from 'handsup-ui'
import 'handsup-ui/src/components/index.scss'

Vue.use(handsupui)
```
